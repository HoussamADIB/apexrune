# Gemini Canvas Design Prompts for ApexRune Website

Use these prompts in Gemini Canvas to get design suggestions for different parts of your website.

## 1. Service Detail Page Design

**Prompt:**
```
Design a modern, professional service detail page for a Salesforce consulting company called ApexRune. The page should include:

- A hero section with a large icon/graphic representing the service
- Clear, bold headline (3rem font size)
- Descriptive paragraph explaining the service
- Grid layout of feature cards (3-4 columns on desktop, 1 column on mobile)
- Each feature card should have a blue checkmark icon and feature text
- A call-to-action section at the bottom with gradient background (light blue)
- Back button to return to services
- Color scheme: Dark blue (#1E3A8A), Bright blue (#3B82F6), White, Light gray (#6B7280)
- Modern, clean design with rounded corners (16px border radius)
- Subtle shadows and hover effects
- Fully responsive design

The page should feel professional, trustworthy, and aligned with enterprise software aesthetics.
```

## 2. Service Tab Navigation Enhancement

**Prompt:**
```
Design an improved service tab navigation system for a Salesforce consulting website. Requirements:

- Five service tabs: Salesforce Quick Start, Custom Automation, Org Health, Platform Integration, Custom Solutions
- Active tab should be highlighted in bright blue (#3B82F6) with white text
- Inactive tabs should have white background with dark gray text
- Smooth transition animations when switching tabs (0.3s ease)
- Hover effects: slight border color change to blue
- Tabs should be responsive: wrap on mobile, horizontal scroll on tablet
- Add subtle icons or indicators to show which tab is active
- Consider adding a subtle underline or background color change on hover
- Modern, clean design matching enterprise software aesthetics
```

## 3. Service Card Content Layout

**Prompt:**
```
Design a service content card layout that displays different service information dynamically. The card should:

- Have a two-column layout: content on left (70%), icon/graphic on right (30%)
- Title should be large, bold, dark blue (#1E3A8A)
- Description text should be readable gray (#6B7280), 1.125rem font size
- Feature list with blue checkmark icons (SVG)
- "READ MORE" link in bright blue with arrow icon
- White background with subtle shadow
- Rounded corners (16px)
- Smooth content transition when switching between services
- On mobile: stack vertically (icon on top, content below)
- Add a subtle fade-in animation when content changes
```

## 4. Overall Website Animation & Interactions

**Prompt:**
```
Suggest smooth, professional animations and micro-interactions for a Salesforce consulting website. Include:

- Page transition animations (fade, slide)
- Service tab switching animations
- Hover effects on buttons and cards
- Scroll-triggered animations for sections
- Loading states for content changes
- Smooth scrolling behavior
- Button click feedback animations
- Card hover lift effect (subtle shadow increase)
- All animations should be subtle, professional, and not distracting
- Use CSS transitions and transforms for performance
- Consider adding a loading spinner for service detail page loads
```

## 5. Mobile-First Responsive Design

**Prompt:**
```
Design a mobile-first responsive layout for a Salesforce consulting website. Key considerations:

- Service tabs should stack vertically on mobile
- Service detail cards should be single column on mobile
- Navigation should convert to hamburger menu on mobile
- Text sizes should scale appropriately (headlines: 2rem mobile, 3rem desktop)
- Touch-friendly button sizes (minimum 44x44px)
- Adequate spacing between elements on mobile
- Footer should stack vertically on mobile
- Images and icons should scale proportionally
- Consider adding a "sticky" header on mobile for easy navigation
- Breakpoints: mobile (320px-768px), tablet (768px-1024px), desktop (1024px+)
```

## How to Use These Prompts

1. Copy any prompt above
2. Paste it into Gemini Canvas
3. Ask for specific design elements, CSS code, or visual mockups
4. Iterate based on the suggestions
5. Implement the best ideas into your website

## Tips for Best Results

- Be specific about colors, sizes, and spacing
- Mention your existing design system
- Ask for code snippets if you want implementation help
- Request multiple variations to choose from
- Ask for accessibility considerations
- Request mobile and desktop versions
