# Flutter Project

A simple Flutter "Hello World" application.

## Getting Started

### Prerequisites

Make sure you have Flutter installed. If not, install it from: https://flutter.dev/docs/get-started/install

### Running the App

1. Get the dependencies:
   ```bash
   flutter pub get
   ```

2. Run the app:
   ```bash
   flutter run
   ```

   Or run on a specific device:
   ```bash
   flutter run -d chrome  # for web
   flutter run -d macos   # for macOS desktop
   flutter run -d ios     # for iOS simulator
   flutter run -d android # for Android emulator
   ```

## Project Structure

- `lib/main.dart` - Main application code with Hello World
- `pubspec.yaml` - Project dependencies and configuration
