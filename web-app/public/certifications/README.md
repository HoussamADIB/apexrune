# Certification Badge Images

Place your Salesforce certification badge images in this folder.

## Image Requirements

- **Format**: PNG or JPG (PNG recommended for transparency)
- **Size**: Recommended 400x400px or larger (square aspect ratio)
- **Naming**: Use the exact names listed below

## Required Image Files

Name your images exactly as shown (case-sensitive):

1. `platform-developer-i.png` - Platform Developer I badge
2. `ai-associate.png` - AI Associate badge  
3. `ai-specialist.png` - AI Specialist badge
4. `technical-architect.png` - Technical Architect badge
5. `integration-architect.png` - Integration Architect badge

## How to Add Images

1. Save your certification badge images
2. Rename them to match the names above
3. Place them in this `certifications` folder
4. The carousel will automatically use them

## Alternative: Using Different Names

If your images have different names, update the `certifications.js` file and change the `image` property for each certification.
